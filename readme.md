Evergreen FastFarmer for Gigahorse
=====

A lite farmer for the Chia Blockchain supporting Gigahorse.

Running
--------

To generate the farmer config:
```
ff_giga init -f FULLNODE_HOST -p FULLNODE_PORT -r FULLNODE_RPC_HOST -o FULLNODE_RPC_PORT -n SELECTED_NETWORK
```

To run the Farmer with TUI Interface(Default):
```
ff_giga
```

To run the Farmer in CLI mode:
```
ff_giga run
```
